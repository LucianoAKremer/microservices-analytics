# ⚙️ Microservices Analytics

Arquitectura de microservicios para el manejo y análisis de datos.  
Incluye servicios independientes desplegados con Docker Compose.

## 🚀 Tecnologías
- FastAPI (Python)
- Node.js + Express
- PostgreSQL
- Docker & Docker Compose

## 📂 Servicios
- `auth-service/` → autenticación de usuarios.
- `data-service/` → manejo de datos (CRUD).
- `analytics-service/` → análisis con Python + pandas.
